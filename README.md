# Babble's Blackjack

## How to Play

This is a simple Blackjack game using SFML. You don’t need to install SFML or a compiler — it’s ready to play!

## Requirements

- Windows PC
- No other installations needed

## How to Run
1. download this repository.
2. Open the `Consale` folder.
3. Double-click `main.exe` to start the game.
4. Enjoy playing!

## Assets
All images, sounds, and other resources are in the `assets` folder. Feel free to use some are custom made Images!
